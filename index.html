App band hone ke baad bhi alert chahiye toh 3 cheezein karni hongi:
1. HTML file ke top me VAPID_KEY bharo (Firebase Console → Project Settings → Cloud Messaging → Web configuration → Generate key pair)
2. firebase-messaging-sw.js file ko site ke root folder me rakho (saath me di hai)
3. functions/index.js Cloud Function deploy karo — isse hi actual push bhejta hai jab stock kam ho.
